Projects

    Seagull Calendar

    Seagull Calendar is a Google Calendar clone. The calendar is a single page app built using Backbone.js, calendar views created without external libraries, and a RESTful back-end built on Rails. Features include: day, week, and month views; email reminders; shared calendars; recurring events; hiding and showing of calendars; event search; click on a day to quickly create events; draggable and droppable events.

    Wavly

     In progress.

    Wavly is a Google/Apache Wave clone that utilizes WebRTC. Wavly lets you create and share 'waves' with other users, all documents are updated real-time. Features include: save versions of waves; view prior wave versions; commit to those prior versions; revert to the most recent wave before the commit; real-time collaborative messaging/document sharing.
